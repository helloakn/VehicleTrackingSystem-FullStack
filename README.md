FullStack Vehicle Tracking System (Include API for Vehicle and API for Admin as well as Dashboard)  

### Built With

* [Next.js](https://nextjs.org/) For Admin Dashboard
* [NodeJs](https://nodejs.org/en/) For admin API and vehicle API
